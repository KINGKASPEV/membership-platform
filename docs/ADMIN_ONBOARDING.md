# Admin Onboarding Checklist

Use this checklist when setting up a new administrator for the platform.

## Pre-Requisites
- [ ] Determine the **Role** required (State, LGA, or Parish Admin).
- [ ] Identify the specific **Jurisdiction** (e.g., "Lagos State" or "St. Mary's Parish").
- [ ] Obtain the admin's official email address.

## Account Creation (Super Admin Task)
1.  Log in as Super Admin.
2.  Navigate to User Management.
3.  Create New User.
4.  Assign Role and Scope (Link to geographic unit).
5.  Set a temporary password.

## Admin Setup (The Admin's First Login)
- [ ] **Log In**: Go to `/login` and enter credentials.
- [ ] **Change Password**: (Recommended) Update password to something personal and secure immediately.
- [ ] **Verify Scope**: Check the Dashboard. Does the "Total Members" count look accurate for your area?
- [ ] **Test Validation**: Find a test/dummy member and try the Validate workflow to ensure permissions are active.

## Access & Security
- **Never share passwords**.
- **Log out** when using a shared computer.
- Report any suspicious activity or unknown admin accounts to IT Support immediately.

## Support
For technical issues, contact: `support@platform.ng`
